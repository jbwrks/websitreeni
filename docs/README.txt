TREENI WEBSITE - COMPLETE PACKAGE
====================================

This folder contains your complete Treeni website with 11 pages.

FILES INCLUDED:
--------------
1. index.html - Homepage
2. style.css - All styling (one file for entire site)
3. about.html - Our Story page
4. esg-leaders.html - Solution page for ESG Leaders
5. procurement.html - Solution page for Procurement Leaders
6. msme.html - Solution page for MSMEs with pricing
7. solutions.html - Solutions overview page
8. industries.html - Industries we serve
9. insights.html - Blog/insights listing
10. case-study-it-services.html - Case study template
11. contact.html - Contact form
12. README.txt - This file

HOW TO TEST LOCALLY:
-------------------
1. Make sure ALL files are in the same folder
2. Double-click index.html
3. Click through all navigation links
4. Everything should work!

WHAT TO UPDATE BEFORE GOING LIVE:
---------------------------------
1. Phone number: Search for "+91 XXXX XXXXXX" and replace
2. Email: Verify "hello@treeni.com" is correct
3. Office address in contact.html
4. Replace "Wipro, Infosys, L&T Tech..." with actual client logos (optional)
5. Add real social media links (LinkedIn, Twitter)
6. Connect contact form to your email/CRM

HOW TO CUSTOMIZE:
----------------
COLORS: Edit style.css, lines 15-20:
  --primary-color: #2B7A4B (change this green)
  --secondary-color: #1A4D2E (darker green)

FONTS: Edit style.css, line 26:
  --font-main: [your font here]

CLIENT LOGOS: Edit index.html, find "Trusted by:" section

NAVIGATION: Edit the <nav> section in any page

HOW TO PUBLISH:
--------------
OPTION 1: Netlify (Easiest - FREE)
1. Go to netlify.com
2. Sign up (free)
3. Drag this entire folder into Netlify
4. Your site is live in 30 seconds!
5. Connect your domain treeni.com

OPTION 2: Traditional Hosting
1. Get hosting (Hostinger, Bluehost, etc.)
2. Upload all files via FTP
3. Point your domain to the hosting
